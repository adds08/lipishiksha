# **App Name**: Lipi Shiksha

## Core Features:

- Practice Sheet Generator: Generate printable handwriting practice sheets with character grids for various languages, including Nepali.  Users can specify the character to be practiced, and the system will create a downloadable/printable PDF.
- Image Capture & Adjustment: Allow users to upload scanned handwriting practice sheets.  The application displays the uploaded image and provides basic image adjustments such as rotate and crop.
- AI-Powered Label Suggestion: Utilize a generative AI tool to generate suggested labels for handwritten characters, speeding up the labeling process. The AI may be incorrect in some cases, requiring user verification and correction.
- Manual Label Verification/Correction: Provide an interface for reviewing and correcting the AI-suggested labels for handwritten characters, and assigning the correct letters to the image sections, with appropriate keyboard layouts

## Style Guidelines:

- Primary color: Blue (#3498db) to represent trust and education.
- Secondary color: Light gray (#f2f2f2) for a clean and uncluttered background.
- Accent: Green (#2ecc71) to highlight important actions and provide positive feedback.
- Use a clear and legible sans-serif typeface (like system fonts) for interface elements and generated sheets.
- Use simple, recognizable icons for actions such as printing, uploading, and editing.
- Maintain a clean and spacious layout to minimize distractions and focus attention on the handwriting tasks.
